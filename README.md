# Forward-Engine-Tester
Тестовое задание для компании "Forward Development"

Задание:
Необходимо разработать консольное приложение, которое проводит тестирование работы двигателя с помощью 2-х тестов:
1) Тест рассчитывает и выводит на консоль время в секундах, которое пройдет от старта двигателя внутреннего сгорания до момента его перегрева.
2) Тест рассчитывает и выводит на консоль максимальную мощность двигателя в кВт, а также скорость вращения коленвала в радиан/сек, при котором эта максимальная мощность достигается.
Рассчитывать результаты тестов точно, аналитическим путем не нужно, интересует получение этих значений методом симуляции. Разумеется, таким образом значения будут вычислены с определенной погрешностью. Плюсом будет возможность контролировать эту погрешность.

Компоненты:

IEngine - интерфейс двигателя, который позволяет создавать различные их типы, объединённые общим поведением. В качестве примера создан двигатель внутреннего сгорания.

EngineCharacteristics - структура, содержащая в себе статические характеристики двигателей, такие как момент инерции коленвала и т.д.

EngineCondition - структура, содержащая в себе динамические характеристики двигателей, такие как текущая температура, ускорение и т.д.

--

MaxPowerTest - тест, показыающий максимальнуню мощность двигателя, согласно его характеристик.

OverheatTest - тест, показыающий максимальное время работы двигателя, до его перегрева.

--

InternalCombustionEngine - реализация интерфейса IEngine, симулирующая работу двигателя внутреннего сгорания.

TorqueFromSpeed - модуль работы с кусочно-линейной функцией, заданной таблицей значений. Для получения промежуточных значений использованна кусочно-линейнвя интерполяция.

--

Forward Engine Tester - главный модуль программы, занимающийся созданием двигателей и запуском тестов к ним.
